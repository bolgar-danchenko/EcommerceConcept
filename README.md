# EcommerceConcept

UI demo for e-commerce online catalogue. 
Built in SwiftUI with MVVM.

### Features:
* Main view
* Detailed product view
* Cart view
* Categories
* Filters

### Stack:
* SwiftUI
* MVVM
* @EnvironmentObject
* @ObservableObject
* JSON Decodable / URL Session
* Custom Navigation Bar
* Custom Shapes with Path
* Adaptive Layout Design
